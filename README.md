# Cutie

Cutie is a fun, enchanting and entertaining humanoid robot head designed to greet, interact with, and engage (and help) people in public events, lobbies, demos, and reception spaces.

This project Cutie is structured as a modular ROS2 robotics system, where sensing, perception, decision-making, expression, motion, and diagnostics are separated into independent subsystems that can be developed, tested, and upgraded individually.
